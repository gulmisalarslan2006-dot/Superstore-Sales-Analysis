#  Sample Superstore - Data Cleaning & Sales Analysis (Power BI)

This project features a comprehensive Business Intelligence (BI) study conducted on the well-known **Sample Superstore** dataset. The primary focus of this project was to transform "dirty" raw data into an analytical format and design an interactive dashboard that provides meaningful business insights.

##Project Overview
This study covers the end-to-end data analytics workflow, including data preprocessing, modeling, and visualization. By applying a statistical perspective, anomalies in the data were identified, and key metrics were developed to support data-driven business decisions.

## Key Implementation Steps

### 1. Data Cleaning & Transformation (Power Query)
The initial dataset contained "dirty" records and inconsistent formats. The following ETL processes were performed:
* **Data Typing:** Fixed incorrectly formatted date and numeric columns.
* **Missing Value Management:** Analyzed null values and applied appropriate imputation or cleaning methods.
* **Text Standardization:** Resolved spelling errors and inconsistencies in region and category names.
* **Duplicate Removal:** Cleared redundant sales data to ensure data integrity.

### 2. Data Modeling & DAX
* Established a relational schema between data tables.
* Developed **DAX** measures for Key Performance Indicators (KPIs) such as Total Sales, Profit Margin, and Order Count.

### 3. Visualization & Dashboard Design
* Created a user-friendly interface with dynamic slicers and filters.
* Visualized regional performance and profitability analysis by product category.
